Mayo Run – Personvernerklæring

Sist oppdatert: 2025

Mayo Run samler ikke inn, lagrer eller deler noen form for personopplysninger.

Datainnsamling

  • Appen samler ikke inn data av noe slag.
  • Ingen brukerdata sendes til utvikleren eller til tredjepartsleverandører.

Analyse og sporing

Mayo Run bruker ikke analyseverktøy, sporing eller annonser.

Tredjepartstjenester

Appen bruker ingen tredjepartstjenester som samler inn data.

Barns personvern

Mayo Run er trygt for brukere av alle aldre og inneholder ikke noe innhold som krever ekstra personvernhensyn.
